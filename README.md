## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Install Node.js
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/marknik139/meal-db-api.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
